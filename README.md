## Features
- Live Bitcoin price in USD
- 24h % change with arrow ↑↓ and color
- Drag anywhere by clicking
- Resize with the bottom-right grip
- Right-click → Copy price or Refresh and EXIT.. . 
- Dark GitHub-style theme
- Always on top
- Updates every 20 seconds
- Uses Coingeckos api
  
No install. No Python needed. Zero bullshit. 

Made for people who just want the price on their second monitor forever.
